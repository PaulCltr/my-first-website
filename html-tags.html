HTML Tag List

Quick reference for basic HTML tags

Home Modules Projects HTML Tags GitHub

This page is a simple reference I created to help me learn and remember common HTML tags while studying TM111.

Structure tags

  • <html> , wraps the whole page
  • <head> , page information and links
  • <body> , visible page content
  • <title> , page title shown in the browser tab

Text tags

  • <h1> to <h6> , headings
  • <p> , paragraph text
  • <strong> , important or bold text
  • <em> , emphasised text
  • <br> , line break

Links and lists

  • <a> , hyperlink
  • <ul> , unordered list
  • <ol> , ordered list
  • <li> , list item

Layout and grouping

  • <div> , generic container
  • <section> , grouped content
  • <header> , page or section header
  • <footer> , page or section footer
  • <nav> , navigation links

Self closing tags

  • <br> , line break
  • <img> , image

These tags do not need a closing tag.

Media

  • <img> , image
  • <video> , video content
  • <audio> , audio content
