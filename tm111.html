A series of structured programming tasks completed as part of TM111, focused on understanding program logic, data handling, and problem decomposition.

What I did

  • Designed and built small programs using Scratch

Tools and technologies

This project strengthened my understanding of how programs process data step by step. I learned how to translate problems into logical instructions and how to test and refine solutions when errors occurred.
